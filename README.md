Proje Hakkında

Bu proje, kullanıcıların günlük beslenme alışkanlıklarını takip edebilmesi için geliştirilmiş görüntü işleme destekli bir mobil uygulamadır. Kullanıcılar yemeklerinin fotoğrafını çekerek yemeğin türünü otomatik olarak tespit edebilir ve buna göre kalori hesaplaması yapılır.

Özellikler
Yemek fotoğrafı ile otomatik tanıma
Görüntü işleme ile yemek sınıflandırma
Otomatik kalori hesaplama
Günlük kalori hedefi takibi
Geçmiş veri ve ilerleme analizi
Egzersiz ve kilo takibi için öneriler sunan sohbet robotu
Sistem Nasıl Çalışır?
Kullanıcı yemek fotoğrafı çeker
Görüntü işleme modeli yemeği tanır
Sistem veritabanından eşleşen veriyi bulur
Kalori hesaplanır ve kullanıcıya sunulur
Veriler kullanıcı geçmişine kaydedilir
Kullanılan Teknolojiler
Mobil: (Flutter / React Native)
Backend: FastAPI / Flask
Veritabanı: SQLite
Görüntü İşleme: Python, OpenCV / Makine öğrenmesi modeli
API: RESTful servisler
Katkım
Görüntü işleme ve yemek tanıma sistemi geliştirme
Backend ve API geliştirme
Kalori hesaplama algoritması
Kullanıcı veri takibi ve analiz sistemi
Sohbet robotu entegrasyonu
Kurulum
git clone <repo-link>
cd calorie-tracker
pip install -r requirements.txt
python app.py
Amaç

Bu proje, kullanıcıların beslenme alışkanlıklarını daha bilinçli hale getirmelerini sağlamak ve günlük kalori takibini pratik bir şekilde sunmak amacıyla geliştirilmiştir.
